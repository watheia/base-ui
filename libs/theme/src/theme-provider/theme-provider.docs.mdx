Adds all the theme context and colors for the project.

This includes:

- Colors
- Headers and paragraphs font-size, margins, etc
- Brand font
- Shadows

```tsx
import { DefaultTheme } from '@watheia/theme.theme-provider';

function ApplicationContext({ children }) {
  return (
    <DefaultTheme className={styles.additionalClasses}>{children}</DefaultTheme>
  );
}
```
